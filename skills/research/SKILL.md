# Research Skill

Research a topic and provide actionable recommendations for a small retail business.

## Instructions

1. Research the given topic thoroughly
2. Identify the top 3-5 options/solutions
3. For each option, provide:
   - **Pros**: Key advantages
   - **Cons**: Key disadvantages
   - **Cost**: Pricing tier (Free, $, $$, $$$) with specifics if available
   - **Best for**: What type of use case it suits

4. Provide a clear recommendation for a small retail business context:
   - Budget-conscious
   - Limited technical staff
   - Needs reliability over cutting-edge features
   - Values simplicity and good support

5. Keep it actionable, not theoretical:
   - Include specific product/service names
   - Mention setup complexity
   - Note any gotchas or hidden costs

## Example Output

```
Research: Payment Processing Solutions

## Options Compared

### 1. Square
- **Pros**: Easy setup, no monthly fee, good POS hardware
- **Cons**: Higher per-transaction fee, limited customization
- **Cost**: Free + 2.6% + $0.10 per transaction
- **Best for**: Small retailers with moderate volume

### 2. Stripe
- **Pros**: Developer-friendly, extensive API, global support
- **Cons**: Requires technical setup, no POS hardware
- **Cost**: 2.9% + $0.30 per transaction
- **Best for**: Online-first businesses with dev resources

### 3. PayPal
- **Pros**: Brand recognition, buyer trust, easy integration
- **Cons**: Higher fees, account holds possible
- **Cost**: 2.9% + $0.30 per transaction
- **Best for**: Businesses wanting quick customer trust

## Recommendation

For a small retail business: **Square**

Why: Zero upfront cost, intuitive POS system, and all-in-one solution.
No technical staff needed. Hardware is affordable and reliable.

Next steps:
1. Sign up at squareup.com (free)
2. Order a Square Reader ($49) or Stand ($149)
3. Download Square POS app
4. Start accepting payments same day
```
